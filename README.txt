A web-scraper-ish script for pasting in the in-browser-terminal. 
(runs directly in browser, no external tools needed,
just access to in-browser-terminal)

Mostly a personal backup.

Currenlty configured to 'finviz',
but theoretically can pull data from any web pages 
(given they share a domain, uses in-browser localStorage,
which is not shared between domains)

The idea is keep running it in terminal, 
runs a script/gives a few options per web page, 
then automatically redirects to next web page.
